# Getting Started

First, make quote data in client application folder.
```
$ cd backend
$ go build -o main
$ ./main
```

Second, install npm library from client source and start server.
```
$ cd client
$ yarn install
$ yarn run dev
```

Finaly, access `localhost:3000`
